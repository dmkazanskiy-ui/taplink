# Тап-линк — Дмитрий Казанский

Лендинг-ссылка для шапки профиля Instagram. Статическая страница, хостится на GitHub Pages.

## Структура
- `index.html` — вся страница (HTML + CSS инлайн)
- `assets/` — изображения (аватар, фон, иконки приложений)

## Как поменять ссылки кнопок
Открой `index.html` и найди комментарии вида `<!-- КНОПКА: ... -->`.
У каждой кнопки замени `href="#"` на нужный адрес.

## Локальный просмотр
Просто открой `index.html` в браузере.
